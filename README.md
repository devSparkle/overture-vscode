# overture-vscode README

Integrates [Overture](https://github.com/devSparkle/Overture) natively with VS Code and [Luau LSP](github.com/JohnnyMorganz/luau-lsp) or [Roblox LSP](https://github.com/NightrainsRbx/RobloxLsp).

The [Luau LSP](https://marketplace.visualstudio.com/items?itemName=JohnnyMorganz.luau-lsp) or [Roblox LSP](https://marketplace.visualstudio.com/items?itemName=Nightrains.robloxlsp) extension is a requirement for several features. **We recommend Luau LSP, as Roblox LSP is deprecated.**

## Features

- [x] Snippets
- [x] Autocomplete *(requires Roblox or Luau LSP)*
- [x] Typechecking *(requires Roblox or Luau LSP)*
